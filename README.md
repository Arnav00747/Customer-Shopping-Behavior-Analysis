# 🛒 Customer Shopping Behavior Analysis

## 📊 Overview

This project analyzes customer shopping behavior using data analytics techniques to extract meaningful insights.

## 🛠 Tools & Technologies

* Python (Pandas, Matplotlib)
* SQL
* Power BI

## 📁 Project Files

* Dataset (CSV)
* Jupyter Notebook (.ipynb)
* Power BI Dashboard (.pbix)
* SQL Queries
* Project Report (PDF)

## 🔍 Key Insights

* Identified popular product categories
* Analyzed customer spending patterns
* Studied purchase frequency trends

## 📊 Dashboard

Interactive Power BI dashboard created for better visualization and business understanding.

## 🚀 Conclusion

This project helps businesses make data-driven decisions by understanding customer behavior patterns.
